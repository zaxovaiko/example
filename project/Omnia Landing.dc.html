import { definePlugin } from '@omnia/core'
import { loyaltyContract } from './loyalty.contract'

export default definePlugin({
  name: 'loyalty',
  contract: loyaltyContract,
  register({ wallet, audit, bus }) {
    // earn points on every settled bet
    bus.on('bet.settled', async (e) => {
      const points = Math.floor(e.stake * 0.1)
      await wallet.credit(e.userId, points, 'loyalty')
      await audit.append('loyalty.granted', { ...e, points })
    })
  },
})

ARCHITECTURE

Built like infrastructure,
not a starter kit.

Every decision optimizes for the same thing: you extend the platform without ever forking it.

01 contracts

Typed end to end

Zod-first contracts compile to an oRPC + Hono API and a fully inferred SDK. No codegen, no drift, no any.

02 extensibility

Plugin host

Everything extends through definePlugin. Register modules, listen on the event bus, override adapters — without forking a single file.

03 topology

Monolith → microservices

Run as one deployable today; split modules into independent services later — with zero module-code change.

04 agents

AI-native by design

Every module ships an AGENTS.md, scaffold commands and an MCP dev server, so agents read, write and extend your codebase safely.

05 reliability

Production built in

Hash-chained audit trail, transactional outbox, idempotency keys, and ports/adapters for every PSP, KYC and aggregator.

06 license

Open core, MIT

Clone the repo, own every line, ship on your own infra. No seat pricing, no lock-in, no black boxes.
